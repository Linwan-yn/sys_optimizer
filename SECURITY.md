# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability within this project, please follow the process outlined below:

1. **Email**: Send an email to [security@linwan-yn.com](mailto:security@linwan-yn.com) with the details of the vulnerability.
2. **Details**: Provide a detailed description of the vulnerability, including how it was discovered, any related code, and any steps necessary to reproduce the issue.
3. **Response**: We will acknowledge your email within 48 hours and will provide a timeline for remediation.

## Security Announcements
We will make security announcements if vulnerabilities are found in the project. These announcements will be posted in the [repository release notes](https://github.com/Linwan-yn/sys_optimizer/releases) and communicated through security advisories.

## License
By reporting a vulnerability, you agree to the project's license. Please do not disclose the vulnerability publicly until we have had the opportunity to resolve the issue.
